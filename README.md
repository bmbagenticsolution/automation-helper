# Automation Helper

A small Android app that runs in the background and drives the UI of any
**other** app on your phone — taps buttons, types text, scrolls, swipes,
takes whole multi-step scripts. Built for testing your own apps; it uses
Android's official Accessibility API.

It also has a **human mode** (on by default): jittered tap coordinates,
curved swipes with variable speed, and randomized pauses between actions
(short most of the time, occasional 1–3 minute "thinking" pauses on the
slow profile) so the activity doesn't show a fixed automation pattern.
